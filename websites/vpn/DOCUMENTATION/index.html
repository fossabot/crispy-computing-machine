A) Introduction - top

Authfy is a responsive App authentication template with modern and clean designed. By using Authfy you can create quickly your app authentication page. It has designed carefully, in order to create a seamless and wonderful user experience all in one page.

Template Features

  • HTML5 and CSS3
  • Responsive Template
  • Built on Bootstrap 3.3.7
  • 1000+ icons (based on FontAwesome)
  • 8 Different Variation
  • Google Web Fonts

B) HTML Structure - top

All page template is a Bootstrap based responsive layout and each page template has been separated by their unique file name e.g login-01.html, login-02.html and so on.. Here is the general structure.

C) CSS Files and Structure - top

Styles

We're using 8 different CSS files that share among each login page template. we have separated styles for improving convenience to edit each template.

Lets take a look closely at template stylesheet files.

  1. bootstrap.min.css

    This Stylesheet contains responsive grid system ,default element style based on bootstrap. It is bootstrap framework main css file

  2. font-awesome.min.css

    Base styles for FontAwesome icon fonts.

  3. login1-style.css

    Import bootstrap and fontawesome core style and contain all custom style for login-01 template in here.

  4. login2-style.css

    Import bootstrap and fontawesome core style and contain all custom style for login-02 template in here.
